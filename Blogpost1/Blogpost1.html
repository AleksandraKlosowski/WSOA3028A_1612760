First Blogpost!

When elaborating on the ideas and ideals of the internet; one could be referring
to its overall vastness of information that appears to be limitless. Whereby all
of the world’s current knowledge is currently stored; and is continuously being updated.
This continuous addition to the internet shows that although the internet appears
limitless, as if to already contain every version of every piece of information.

When looking at the Library of Babel by Jorge Luis Borges, this is suggesting that
within this library, any possible word, phrase or combination within the English
language exists. This is done through multiple possibilities being created by trying
out every possible variation created with the letters of the English language; showing
us that every possible phrase, sentence or word already exists. This is similar to
the internet by expressing its limitlessness and how there can never really be truly
new information; even more so through the fact that one can actually google the library
of babel and type any word/phrase on this page and will undeniably found that it will
exist within this library. Scientists for example are continuously learning and expanding
on their studies and thus contributing to the added information within the internet;
however due to the limitlessness of the library of babel, ironically these new
discoveries should already be within the library of babel since it contains every
possible formula within the English language. Therefore, if the library of babel is
already on the internet, there really can never be any new information, but rather
exposed information that can already be found within this library.

When looking at The Garden of Forking Paths by Jorge Luis Borges, one is looking at
an elaborate maze, a labyrinth of sorts. It is said that the book itself not only
explains the philosophy of the vast variations of storylines but is also a maze itself,
with its vast and endless interpretations. This is whereby a lot of stories and versions
of a narrative exist simultaneously, as if to express vast timelines that could branch out
dependant of ones choices, much like ones experience on the internet. Similarly the internet
is a labyrinth; whereby depending on your keywords and choice of search, you will find various
different pages and have a vast access to different types of knowledge. Just like the vast
choices and routes one can take within a labyrinth; one similarly leads themselves through
various paths on the internet dependant on what they click on, and where they try gain
access to. One could end up with a nasty virus, or have successfully downloaded their
pirated movie. The internet is vast and is expressive of how large the world of information
and misinformation. As much as the world allows us to gain access to multitudes of studies
and knowledge, it can easily deceive the public due to the fact that it equally contains
false information.

The main feature that perhaps makes the internet appear smaller and more censored is
due to algorithms that will pop up the most searched sites and pages; where it will
filter out the less desired information that one can find by sifting through various
pages such as within a Google search.

BlogPost 3!

1. Semantic Markup - what is it and why is it necessary
• Semantic markup is a particular way in which one will structure and write
their HTML coding in order to reinforce the division of etymology and logic
involved with meaning. This in particular has to do with the content of the HTML
as opposed to appearance. This is important as it allows one to navigate easily
through their code and make effective changes when necessary. It is also particularly
helpful when working with a team as it allows for others to navigate and understand
your code within its content; essentially making it easy to distinguish between
different types of data.

2. What is a browser? How do they work?
• A browser is a computer program that contains a graphical user interface for
the use of presenting and navigating between various web pages. More specifically
it is an application software that provides a user with access to the World Wide Web.
This application/browser retrieves wanted and searched for content from a web server
and allows for the user to view it on their particular device whether it be a cell phone,
a laptop etc.

3. What are URLs and how do they work?
• A URL or ‘Uniform Resource Locator’ is the address of a web page usually
beginning with https:// for trustworthy and secure web pages. It is a reference
to a web source that helps the user identify its location within a computer network
and is a means of retrieving it. How it works is that it is a set of directions that
can be interpreted through various platforms and they ultimately point and direct to
a resource. This is done through a code-like language that consists of a protocol, a
subdomain, a root domain, a top-level-domain, a subfolder, a file and a parameter;
ultimately allowing the browser to navigate to the desired page.

4. A reflection on McLuhan, M. ‘The Medium is the Message’
• Mcluhan suggests that the class of a medium is vastly more significant than the
meaning or substance of the message. He states that a ‘message’ is “the change of
scale or pace or pattern” that it “introduces into human affairs.” Personally I agree
with this statement as it provides the readers with an abundance of knowledge and
communicates information. It informs us about what is happening globally and around us.

5. A reflection on Wardrip-Fruin, N. ‘What Hypertext Is’
• Wardrip-Fruin Nelson looks particularly at definitions constructed by Theodor Holm,
the definitions being “A File Structure for the Complex, the Changing, and the
Indeterminate” and “‘Hypertext’ means forms of writing which branch or perform on
request; they are best presented on computer display screens... Discrete, or chunk
style, hypertexts consist of separate pieces of text connected by links.” (Holm in Nelson pg 127)
This is particularly true as hypertext overall is when text is displayed on the screen
of a computer or alternative electronic device and references other texts that can be
accessed instantly by the reader.

BlogPost Number 4!

1. How does the Medium is the Message impact our understanding of the web?

Marshal McLuhan’s work, The Medium is the Message demonstrates to readers how artefacts as media
have the capability to influence any society by their attributes or substance.

It essentially influences and provides clarity for us, the web designers as to how we can approach
the design of the web in the most appealing and functional way possible. It allows us to make
thought through and educated decisions on what medium would be best suited for our audiences and
intentions. For example; one would look for the most efficient browser that has sufficient speed
in order to create a more desirable and user friendly experience.

The browser would also need to be well known, or well-advertised in order to encourage users or to
receive an adequate flow of visibility whereby users can easily find our webpages. It also helps us
understand user experiences through different mediums such as through film versus the web, as film
is more linear, whereby users participate and experience it from beginning to middle to end; however
the web is seldom experienced through a linear experience, the user is in more control as to where
they want to go, what they want to see and ultimately can navigate directly to what they came for.
It is entirely up to the user how they decide to experience and consume the information being provided.

Another thing whilst looking at medium is accessibility. One can also look at the web not only as a
medium, but as a message: users can now access the web globally and from anywhere within the comfort
of their own home; unlike newspapers that prove to be far more restrictive whereby readers will have
to wait for delivery and accessibility is far more circumstantial; however I this day and age, one
can gain access to news instantly, as soon as the article has been publishes. Through the use of
computers and cell phones, one can also dictate the brightness and use filters such as the ‘blue light’
filter for further comfort.

Most importantly, web designers have the capability to produce tools, experiences, services and
societies due to the medium that is being used. Historically, no other medium to date has the
capability to reach as many individuals on such a global scale. It forces us designers to account
and keep in mind its accessibility, reach and immediate gratification.

As designers we need to look at the best means of delivering information dependant of the job or
assignment given. Through the use of the web, a small store can gain far more coverage and an
access to a wider group of clients; however in a small town of mostly elderly individuals that
are not technologically sufficient, the newspaper may be a greater form of communicating to
potential, nearby clients.

My FINAL blogpost!!! (For Now)

The importance of Accessibility

The web has become the most popularized form of medium, giving us access to vital
resources that comprise of all the aspects within our lives. The web allows us to
gain complete accessibility to various areas such as healthcare, schooling, looking
for jobs, shopping online, entertainment and much more. Therefore it is vastly
important that everyone has access to the internet.

Having access to the web can assist various types of individuals such as those with
disabilities and essentially can improve life’s experiences for everyone. The
internet also allows us to find the easiest means of communication and allows us
to communicate with others. This aids people who cannot physically go to a shop or
struggle to read on more traditional mediums such as print.

Arguably, the most vital and often overlooked aspects of businesses often include
the lack of reachability for new customers due to a poor implementation of accessibility.
A businesses success can be greatly affected by the usability and accessibility of a website.
If customers cannot navigate through a website, it can be difficult to find the desired
services or items. It can also potentially irritate and infuriate potential customers,
often resulting in lost business. Furthermore, an inaccessible website can leave potential
for legal issues.

Often a websites accessibility is dependant of its user interface and its respective
structure and design. Users should be able to search using keywords; and more specifically
for businesses, have an ease at contacting the business should there be any questions or queries.
The website should be structured in an aesthetic manner whereby many different types of clients
have a pleasant experience, whether it be the elderly or a teenager. This is done through the use
of headings, colours, icons, legible text, images, focus etc.

The use of menus is also incredibly important. Menus should be easy to navigate through
by using keywords of the most important webpages and accesses within the website, as only
providing a search bar can be confusing and may result in a customer being unable to find
desired items as well as being able to have a knowledge of what is being offered. It is
through a websites functionality that develops a certain need and desire to use it.
Functionality should be all inclusive in order for a wider sense of engagement. This can
also be done by allowing the user to select a particular language in which one could display
the webpage; there could also be an option that allows a shopper to select the country with
which they reside which could show what is offered and available within their region; a further
bonus would be converting the prices into the consumers currency for ease.

All in all, accessibility is one of if not the most important access to any type of user
and business.
